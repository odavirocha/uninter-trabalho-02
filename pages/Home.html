Sobre mim

Eu me chamo Davi, nasci em Jaguaruana, no interior do Ceará, mas minha família se mudou para Porto Alegre, no Rio Grande do Sul quando eu tinha dois anos, onde morei por 18 anos, até que em 2022, nos voltamos para o Ceará, onde atualmente moro em Fortaleza.

Sempre gostei muito de jogos e foi por conta deles que tive meu primeiro contato com programação, mas não foi o suficiente para eu seguir carreira. Já trabalhei como Designer, criando posts para Instagram, e também como editor de vídeos curtos para o Tiktok. Além de jogar gosto de ler e e ir a praia.

Profissionalmente, estou estudando para ser Desenvolvedor Backend com foco em Java e Spring Boot. Tenho experiência prática na criação de APIs REST seguras utilizando Spring Security com autenticação JWT completa (incluindo tratamento de filtros e controle RBAC). Na parte de dados, trabalho com PostgreSQL e MySQL gerenciados por Flyway e ambientes conteinerizados com Docker. Garanto a qualidade do meu código através de testes com JUnit 5 e Mockito, além de monitoramento com Spring Actuator.

Formação

Engenharia de Software - Uninter

Formação focada no desenvolvimento de sistemas modernos e seguros, englobando linguagens como Java, Python e tecnologias web, metodologias ágeis, segurança cibernética, testes de software e simulação de projetos reais.

Object Oriented Programming in Java - IBM

Domínio dos pilares da orientação a objetos na prática, manipulação de fluxos de dados (Streams) e gerenciamento de arquivos com o ecossistema Java.

Database Management with Java and SQL - Amazon

Integração prática de aplicações Java com bancos de dados relacionais via JDBC, focando na escrita de queries SQL e manipulação segura de dados.

Oracle Java Foundations - Oracle

Consolidação da sintaxe base e estruturas de controle do Java até conceitos fundamentais de POO e execução em nuvem.

Foundations of Cybersecurity - Google

Focado nos pilares da segurança da informação, análise de vulnerabilidades, frameworks de defesa e introdução a ferramentas como Linux e SQL.
